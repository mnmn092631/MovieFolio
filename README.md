## 실행 방법

```txt
// .env

AUTH_URL=
AUTH_SECRET=

AUTH_GOOGLE_ID=
AUTH_GOOGLE_SECRET=

DATABASE_URL=

// 영화진흥위원회 오픈API
MOVIE_LIST_BY_TITLE_API=영화 목록&movieNm=
MOVIE_INFO_BY_ID_API=영화 상세정보&movieCd=
MOVIE_BOX_OFFICE=일별 박스오피스&targetDt=
```

```sh
npm start
```
